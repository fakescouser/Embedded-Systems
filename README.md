# Embedded-Systems
An STM32F722xx and Zephyr OS embedded health monitor. It fuses data from MAX30102 ($SpO_2$), DS18B20 (Temp), ADXL345 (Accel), and GSR sensors. Features a local TFT LCD, 16GB SD logging, and a Flutter companion app. Features ESP8266 Wi-Fi connectivity to stream real-time biometric telemetry directly to a remote NoSQL database.
